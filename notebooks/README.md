#  Notebooks

This folder contains Jupyter notebooks used for the complete data analysis workflow.

---

##  Contents

* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA)
* Visualization of trends and patterns
* Behavioral analysis of traders
* Insight generation based on sentiment

---

##  Key Workflow

1. Load datasets
2. Merge trading data with sentiment index
3. Perform EDA (visual + statistical)
4. Analyze behavior across:

   * Fear phases
   * Greed phases
5. Extract meaningful insights

---

## Best Practices Followed

* Clear step-by-step workflow
* Proper comments for readability
* Minimal unnecessary outputs
* Logical flow from raw data → insights
